# 肾（膀胱）


## 肾当令时辰（申时膀胱经，酉时肾经）
膀胱经当令时辰是下午3-5点（申时），肾经当令时辰是下午5-7点（酉时）。
::: tip 时辰病
下午3-5点容易困倦的，或是其它什么毛病，治膀胱。
下午5-7点爱出的毛病，治肾。
:::

::: tip 巧补肾
每天下午6点吃6颗黑豆，补肾。这是取巧用数补肾，类似于一个支点撬动地球的巧力。
:::


## 肾主脑——肾主髓，脑为髓之海

脑子出现一切问题，都是肾虚。<br>
脑子里不管长什么东西，都在腘窝处放血。

::: tip 案例：脑肿瘤
这是一个左常波老师的案例。一老太太，因脑肿瘤压迫视神经，看不清了。<br>
左老师在腘窝处放血。周老师猜测她腘窝处一定有明显瘀络。放血之后，在没有人为因素的情况下，血一直流，放血过程中，医者几次欲止血，患者说别止，我已经能看清东西了。一直放了600CC血，血才自行止住。这是周老师听说过的放血量最大的一次。（注：600CC，就是一瓶脉动）<br>
能看清东西了，说明肿瘤在变小。这一针放血，起码好了60%。<br>
经验总结：人体有自控能力，当脏血放完了，它会自己关闭，不用人为止血。
:::

::: tip 案例：脑血栓、脑瘀血
治疗方案：先吃药再耳朵拨罐。93年一河南老头来北京专治脑血栓。治之前他先看病，看眼睛，他就能知道你是不是有脑血栓会不会得脑血栓，没有的他不拨。药是他特制的，吃药后一个小时就拨罐。在耳朵周围用湿面团密封，能拨出很多大黑血块，拨完就能走路胳膊就能打直了。一周后再拨另一边，拨完就彻底好。也就是两次治疗后就能好完。拨之后耳朵肿得像猪头。<br>
这老头以前在河南一医院给人治，后来医院院长学会了，但没学到药怎么做，只学到了耳朵拨罐，能治好50%的患者。<br>
方案依据：肾开窍于耳。耳是肾的通道。
:::

::: info 脑血栓有哪些症状？
视觉模糊、昏暗： 视力模糊是脑中风的早期症状之一。<br>
言语不清： 说话困难 是即将发生脑卒中或脑卒中的另一种常见症状。<br>
麻木： 面部两侧长时间麻木可能是脑中风的征兆。<br>
急性瘫痪： 脑血栓可能导致患者身体两侧部分瘫痪。<br>
行走困难： 脑中风会导致患者失去平衡或失去四肢的协调性。 
:::

## 肾主骨，久立伤骨劳于肾

骨质疏松，是肾的问题；缺钙，是肾的问题。补钙没有用，因为是肾的吸收出了问题了，得补肾。阴虚补阴，阳虚补阳。<br>
头上的问题，是肾阳不足，补肾阳；如果上火了，补肾阴。<br>
骨骼出现的一切问题，如膝盖疼、腰疼，都是肾的问题，肾虚。

::: tip 案例	听宫附近关节痛，张不开嘴（西医叫XXX神经炎）
病因分析：肾的问题。肾经的分支末稍在两颐。<br>
治疗方案：揉宗筋，吃金匮肾气丸，吃了一年，吃的过程中好的，也不知道具体是吃了多久好的（因为她吃金匮不是为了治这个病，她是为了治例假不来）。后又吃了八个月，上火了，改吃六味地黄丸。<br>
该案例详见4（7）女子以肝为本。
:::

::: tip 案例	骨折
治疗方案：绝不能用消炎药！！！消炎药都是寒凉性质，与骨折加在一起，对肾来说就是雪上加霜。一骨折病人，输液消炎药十五天，成了半身不遂，见前一节。<br>
正确做法是，吃活血化瘀药，加补肾。<br>
经验	笔者祖传经验，骨折之后，不吃任何药，一天一根丝瓜，骨能很快长好。笔者亲见自己外祖母，分别在七十和八十骨折过两次，都是大腿骨折，都是一天一根丝瓜，吃了大约一个月就完全好了，每次骨折后都是几天就下地走路了。
:::

::: tip 案例	牙长不停（溢髓症）。
这是李时珍的一个病例，老长牙，又痒又疼，难受，老得啃东西。<br>

::: tip 案例	膝盖痛
【一】肾引起的。肾经、膀胱经都是盘绕膝盖一周而过。特征是上下楼痛，因为所有重量都承在一个膝盖上，骨骼不好就承受不住。这是肾阳不足引起的。一38岁女，膝盖痛腰痛，来按摩，周老师让回去吃金匮肾气丸，不用按摩了，她补了一年才补上来。<br>
肾引起的膝盖痛是最难治的，得先吃药补肾，然后再配合按摩，才能好。按摩有一个要点，白环俞穴，治肾引起的膝盖痛，特效。<br><br>
【二】肝引起的。肝引起的膝关节痛，特征是关节咯咯作响。因为肝主筋，是筋拉扯引起的痛<br>
治疗方法：揉宗筋<br><br>
【三】心引起的。由心脏引起的膝盖痛，特征是坐了之后站起来痛。因为诸痛痒疮皆属于心。小肠经的下分支沿大腿前面正中央下行绕膝盖打圈然后与胃经合并继续下行。所以50%的膝盖痛是心脏病的前兆。<br>
治疗方法：揉脖子。董氏心膝穴，董氏肩中穴（特效）。<br><br>
【四】脾引起的。由脾胃引起的膝盖痛，特征是肿。因为脾胃经打圈经过膝盖。<br>
治疗方法：治肚子——中脘（胃募，腑会），腹股沟，伏兔，跨骨沿儿<br><br>
【五】肺引起的。季节气候变化，或者人为气候变化，或者吃了冷饮，引起的膝盖痛，这是肺的原因。因为肺主至节。<br>
治疗方法：大肠俞，天枢（大肠募穴）。这是两大要点。
:::

::: tip 笔者案例  膝盖痛
右侧躺，如果左脚在右脚前，左膝盖挨床，大约三五分钟后，抬起左脚时左膝盖就会很痛，要缓上几分钟才能缓过来。<br>
病因分析：这是心引起的。小肠经经过髀关时有瘀堵。<br>
治疗方案：揉左腿髀关，重点是腹股沟韧带下沿偏外侧，揉的时候酸痛，揉了两次后，酸痛减轻，膝盖就再也没有痛了。
:::


## 肾开窍于耳及二阴，其华在发

老年耳背、听力减弱、耳聋，都是肾虚。<br>
50岁后，必须天天补肾，不能间断。阴虚补阴，阳虚补阳。补足肾，老年就不会耳背。<br>
每天下午六点吃六颗黑豆，以术补肾。天一生水，地六成之。<br>
黑豆黑米黑芝麻，以黑豆为首。前面用老虎钳子钳针那位老太太，坚持了四年（每天下午六点吃六颗黑豆），补肾效果非常好，本来精瘦一老太太，长了二十斤肉。

::: tip 案例   男孩耳背
杨维杰在济南讲课，周老师去听课。新疆一6岁男孩耳背，听力只有正常的三分之一，父母带着来找杨维杰求治，杨不给扎，他扎不好，说让周老师治。周老师判断他是先天性的，问妈妈是不是没怀孕前就老腰疼，答是。周说这是贫土长不出好庄稼，猜测孩子睾丸小。一摸，果然如花生米般大，比同龄孩子小一半。<br>
治疗方案：捏睾丸。让妈妈每天捏。睾丸为外肾。周老师说你什么时候给他捏到正常大小了，他就好了。另外多吃黑色食物。三个半月后，给周老师寄来一箱干果水果，说明治好了。
:::

## 肾为胃之关

胃有一个门，这个门就是肾在管。这个门关不住，人的表现就是呕吐；门关不好，人的表现就是呛；门关死了，人的表现就是胃胀胃鼓。<br>
脑瘀血、脑血栓，犯病时喷射式呕吐。治好后留个后遗症，喝水呛，吃东西呛。这些都是肾的问题。

::: tip 案例	喝水呛，吃东西呛。
问周老师能治不？周老师答能治<br>
治疗方案：扎肾关、太溪，一次就好。
:::

::: tip 案例	胃鼓
一女减肥，周身都减了，就是胃还鼓老高。<br>
病因分析：胃门关死了，肾的问题<br>
治疗方案：趴下，腘窝，梅花针敲后拨罐，血一出来胃就不鼓了。
:::

::: tip 案例	胃胀
周老师一同学，胃胀得不行，先扎针，扎完还胀。趴下，腘窝，梅花针敲后拨罐，血一出来，同学说感觉肚子里的东西就掉下去了。
:::

::: info 小结	
注意：这里胃胀胃鼓的腘窝放血，与前面脑肿瘤的腘窝放血，是不同的放血方法。<br>
胃胀胃鼓的腘窝放血，是梅花针先敲再拨罐（很痛）；<br>
脑肿瘤的腘窝放血，是在腘窝处找瘀络三棱针刺络放血。
:::

::: tip 案例	心如悬若饥状
心悬，老感觉饿  或  心不悬，老感觉饿<br>
《一针疗法》中记载，一人心如悬若饥状，按心脏病治了十二年没治好。扎太溪，一针就好。<br>
病因分析：是肾的问题，不是心脏的问题。<br>
治疗方案：扎太溪。一针就好。<br>
方案解析：老饿，是肾阳亢，扎太溪，激发肾水，致阴阳平衡。<br><br>

另，周老师的一个山东学员，治了一个心如悬若饥状，一针太溪，立即好。<br>
另一个，老感觉饿，心没悬，也是一针太溪就好了。
:::

::: info 总结	
胃和肾关系很密切，治胃不治胃，治肾；治肾不治肾，治胃。<br>
能吃能喝，不能算胃好。胃好不好，要看能不能抗饿。胃好的人，饿两小时没事；糖尿病人，饿十分钟就浑身哆嗦虚汗难受。周老师在给人按摩背部时，常常发现人的胃不好，但人都不自知。
:::

::: info 笔者思考
“肾阳亢，扎太溪，激发肾水，致阴阳平衡”，这句话，表明了周老师治病的基本思路。<br>
阳多了，就升阴，致阴阳平衡；<br>
阴多了，就升阳，致阴阳平衡。<br>
这样的做法，是把人体系统环境一步步往更好的级别提升。<br>
如果一上火就吃清热去火的药，这个思路是减少阳使之与阴平衡，同样是阴阳平衡了，阴阳一平衡症状就没有了病就好了，但人体系统的整体阴阳小了一圈，系统环境也就降了一级。
:::


## 诸寒收引皆属于肾

所有进入身体的寒，如果没有得到正确地治疗，最后都会进入肾收藏起来。<br>
身体里的一切寒，不论是胃寒，还是肺寒，最后都入肾，所以肾上的病，90%都是因为寒凉引起的。<br>
所有的发烧，都是因为寒，是内阳和外阴相博的表现，一定不能消炎退烧，因为消炎药都是寒凉的，只会助长寒凉，相当于资助敌军；而退烧相当于撤我军，所以消炎退烧都是在跟身体系统捣乱，只会加重病情。这时候要助阳，阳一上来，就把外阴驱走了，烧就退了。助阳的方法可以是艾灸，也可以是大热药，如附子。<br>
凡是寒凉，都是肾的问题。所以治所有的发烧，都是在督脉上（大椎、身柱、陶道、百会）艾灸，治的是肾寒。<br>
养生要穴，也是灸身柱陶道，目的也是袪肾寒。<br>
治病，一定不能忘了阴阳表里寒热虚实。

::: tip 案例	尿血
病因分析：夏天每天喝冰镇绿豆汤，喝了四年，胃痛了，治好后，又得了尿血。其实引起胃痛的寒并没有走，而是转移住所了，进肾了。<br>
治疗方案：尿血了三年，均按热重治，没治好。因为治反了。不是热而是寒<br>
吃金匮肾气丸，白茅根熬水喝<br>
注：白茅根止血、利尿、抗炎。白茅根，白色入肺，补的是肺，虚则补其母，正合补肾原理，所以它治的是肾的毛病：止尿血、抗尿道炎。
:::

::: tip 案例	17岁强直性脊柱炎
浑身疼，发高烧。西医治了一周，没退烧，转中医科，又治一周，还是没退烧，因为西医中医都用消炎药，所有的消炎药都是寒凉的，治反了，只会加重病症，不会治好。<br>

::: tip 案例	睾丸炎

## 肾气虚则厥，实则胀、五脏不安

厥，就是反，就是和正常相反。<br>
中医中一般用厥字表达身体的两种非正常状况：<br>
【1】	难受得要死，但又说不上来具体怎么个难受法。<br>
【2】　	冷。通常说厥逆、厥冷，或直接简称厥。

::: tip 案例  浑身难受
60多岁老太太，浑身难受，一到下午更难受，又说不上来究竟怎么个难受法，天天给三个孩子打电话说要死了。医院诊断为癌，但又不知道是哪里的癌。求周老师治，周老师不给治，说我哪里会治癌呀。求了两个月，周老师答应看看。来了一看，周老师乐了，这不就是书上说的肾气虚则厥吗？可算见着厥是什么样的了：就是难受得要死，但又说不上来具体怎么个难受法。厥，就是反，就是和正常相反。<br>
所以这是肾气虚。具体是阴虚。当肾阴虚到相当程度时就会厥。<br>
治疗方案：周老师给按摩了两次，头天正面揉一次，第二天背面揉一次，好多了。嘱咐回去吃六味地黄丸，吃了一年多，活蹦乱跳的。
:::

::: info 小结	
肾虚要分清阴阳，阴虚是厥，阳虚是腰腿痛
:::

## 如何判断肾阴虚还是肾阳虚

一般的认识是，人年老阳衰，自然是需要补肾阳。补肾阳，自然是吃金匮。而长期吃金匮，就会导致掉牙。这是编者自己用数年的时间亲测出来的结果。编者丈夫，从大约42岁开始吃金匮，吃了大约两年多，牙就开始松，越来越松，然后开始掉。知道牙松牙掉都是肾的问题，于是更加认真地吃金匮。但是牙仍然松、掉，牙一松，吃饭就慢，越来越慢，不仅如此，还时不时地来一次牙龈肿痛，都以为是因为牙松，渣滓落入牙龈缝所致。最严重的时候是门牙掉了一颗，左边掉了一颗松了两颗，右边掉了一颗松了一颗，简直没有办法吃饭了。跑了无数趟牙医，也只能把门牙种了，松的也没法。大约从2024年年初开始，就把金匮停了，停了一两个月，开始吃石斛夜光丸，大约吃了三周，吃饭速度明显加快了，但当时没注意到这一转变，后来看周老师说的每天下午六点吃六颗黑豆的方法，于是把黑豆炒了吃，吃了几周忽然意识到以前的松牙怎么可能嚼得动炒黑豆？这才关注到当前的情况，不知从什么时候开始，牙已经完全好了，松的都稳固了，也没有牙龈肿痛了，吃饭速度也正常了。<br>
长期吃金匮，还会导致尿急漏尿症状加重，这也是编者自己用数年的时间亲测出来的结果。初时有轻微漏尿，是咳嗽或打喷嚏时漏一点点，吃金匮两三年，渐渐漏尿越来越严重，还关不住，到来不及跑厕所的程度。后来改吃石斛夜光丸，一个多月就完全好了。<br>
分析：金匮不是阴阳并补，而是补阳之力大过补阴之力，长期下来，就会导致阴长速度不及阳长速度而出现水被烧干的情况。肾阴消耗太过，就出现松牙掉牙症状，而且尿频尿急漏尿这些症状也会加重。<br>
为什么肾阴虚耗太过，就会出现松牙掉牙，以及加重尿频尿急漏尿这些阳虚症状呢？这要从肾的工作原理说起。人是一个移动电器，肾就是移动电源，也就是电池，电池之所以能贮电，是因为电池液，肾阴就是电池液，肾阳就是电。人每顿吃饭，身体都会把食物转化为电能存入肾里，所以肾里的贮电多少，取决于肾里的电池液的多少，电池液多，贮电就多；电池液少，贮电就少；电池液没了，电也就没法贮了。所以出现的一切肾阳虚的症状，都不是因为没电，而是因为没有电池液，电发出来没有地方贮存，人体发电的能力是极强的。所以一切肾阳虚的症状，只要补肾阴，让电发出来有地方存，肾阳虚的症状就会改善。<br>
那么什么时候用金匮？急症的时候救急用，比如尿急漏尿脱肛子宫脱垂这些阳虚症状，吃金匮，症状改善了，就转为补肾阴；还有痛经、尿血、便秘等寒症，用金匮，症状改善了，就转为补肾阴；周老师还有一个案例，就是先天不来例假，这种情况下，必须用附子把她的先天肾阳调出来，所以用金匮，案例中用金匮用了一年八个月，直到她牙龈肿了，才转为补肾阴。<br><br>
结论：<br>
【一】	但凡是肾虚，不论症状是阴虚还是阳虚，归根结底都是阴虚，都应补肾阴；<br>
【二】	金匮不是用于滋补的，它是用于肾阳虚的时候治病或救命的，只能用一时，不能长期用。

知识扩展：金匮肾气丸和六味地黄丸<br>
六味地黄丸，由金匮肾气丸方裁减而成，由地黄、山药、山茱萸、茯苓、泽泻、丹皮六味中药组成，故名六味地黄丸，专事滋阴。方中三补三泻，补而不滞，滋而不腻，是最适合长期食用的补肾药方。<br>
金匮肾气丸，出自《金匮要略》，又名八味地黄丸，是前面六味的基础上，加了附子、桂枝而成，仲师的本意是遇有肾阳虚的病人，助其壮阳，又不忘培补其阴，免致阳太过而阴太损。所以金匮长期用就是会导致阳过而阴损的。仲师的本意也是金匮是用于治病救命而不是用于滋补的。<br>
附子，并不是补阳的药，而是取肾阳的药，它是把肾中的备用电取出来用。正常情况下，肾的备用电是不可用的，只有附子能取。所以附子有救命回阳之功。举凡人有危殆重症的情况，肾中普通电已不够支撑，就可以用附子取用备用电。但备用电是只可用不可存的，用一个少一个。备用电是天生自带的，主要用于造精繁衍。<br><br>
对于阳虚和寒症，补阳的方式只有两种：<br>
【一】	灸：包括天灸（晒太阳，主要晒背）、艾灸（陶道、身柱、大椎）<br>
【二】	动：包括按摩（搓八髎点长强）、运动（八段锦、散步）
