# 脉疗法————心、脑

<br><br>
脉疗法和宗筋疗法，是周老师最常用的两大疗法。<br>
操作手法：揉脖子。顺筋为揉，横筋为拨，这里是揉。<br>
脖子对应区：<br>
中间（咽喉部） - 肾。<br>
右边 - 肝。治妇科病、阴道痒很有效，因为肝经环绕阴器，下病上治。<br>
左边 - 心。
<br><br>

::: tip 案例一	
风湿性心脏病 - 浑身疼了一个多月<br>
治疗方案：揉脖子。20分钟，就不疼了。<br>
:::

<br>

::: tip 案例二	
膝盖痛，脚腕肿<br>
病因分析：50%的膝盖痛或脚腕肿是心脏病的前期反应。<br>
治疗方案：揉脖子。
:::

<br>

::: tip 案例三	
胃病<br>
治疗方案：治胃不治胃，治心包，因为胃与心包是别通关系。所以也是揉脖子。
:::
