# 心（小肠）


## 心当令时辰（午时心经，未时小肠经）
心经当令时辰是中午11-1点（午时），小肠经当令时辰是下午1-3点（未时）。
::: tip 时辰病
中午11-1点出现的毛病，治心。
下午1-3点爱出的毛病，治小肠。
:::


## 舌为心之苗

舌尖为心，左为肝，右为肺，中为脾，根为肾。<br>
按摩前一定要先看看舌。心脏很不好的，经不住捏揉，一捏就导致心脏病卒发死亡。<br>
舌头吐不出来，舌尖是平的，舌尖中间有一沟，都是严重的心脏病的表现。<br>
舌头歪，是大脑毛细血管血栓。

::: tip 足底按摩与心脏病卒死的关系
那些做足底按摩的，都喜欢一来就使劲按心反射区，心脏是不喜欢的。两人喝了酒，说去按摩足底放松放松，结果一按就都当场死了。因为喝了酒心脏负荷本就大，再一使劲按心反射区，心脏承受不了，就直接死亡。然而现在评判以西医理论为准，西医理论并不认为心脏病卒死与足底按摩有任何关系，所以足按摩师并无获罪，但实际就是足底按摩直接导致死亡的。<br>
<br>
足底的心反射区，位于左脚脚掌中部偏外侧，即在左脚第4、第5跖骨之间，与肺脏反射区、脾反射区相邻。<br>
                右脚同样位置，是肝反射区。
:::


## 心开窍于耳，耳兼心肾（水火即济）

::: tip 案例一	耳鸣
病因分析：肾水不足，心火上炎<br>
治疗方案：补肾泻心。补肾水——六味地黄丸；泻心火——拍打两肘。<br>
吃六味地黄丸一个月之后，再拍打两肘。如果一次没好，过一周再拍。
:::

::: tip 案例二	中耳炎，化脓，痛甚
病因分析：痛，是心；耳，是心和肾。<br>
治疗方案：放血（肘部放血或中指、小指四缝放血）<br>
放一次就不痛了，然后就补肾。<br>
那么补肾是补肾阳还是补肾阴？<br>
本案例中周老师没讲。笔者认为，根据周老师前面讲的，一切炎症都是寒症，所以应该补肾阳：金匮肾气丸。
:::

::: info 思考一
习惯了西医治病思维的人都会想，怎么不消炎不杀菌呐？不消炎不杀菌中耳炎能好吗？<br>
中医思维是这样的：<br>
首先，炎症是由细菌感染引起的，这个大家都是共识。中医关注的是，细菌无处不在，为什么昨天没感染你今天感染你了？为什么没感染他感染你了？为什么没感染你的眼睛或者口腔感染到耳朵了？<br>
这说明，昨天你的身体环境还不适合细菌感染，今天适合了；今天你的眼睛或其他部份环境不适合细菌感染但耳朵适合了。所以问题在于耳朵而不在细菌。要治的是耳朵而不是杀菌。<br>
耳朵归谁管？心和肾。耳朵环境不好才会招来细菌，说明心和肾环境出问题了，所以放血（改善心环境）、补肾（改善肾环境）。<br>
心和肾的环境一旦恢复好了，细菌自然就飞走了，好比你把臭鸡蛋丢了，苍蝇自然就飞走了一样。所以在面临炎症时杀菌根本不是我们该做的事。杀菌，不仅不会帮助到我们的身体，反而是在捣乱，因为杀菌药都是寒凉的，而攻击人体的细菌也都是寒性的，对身体来说，它们是同一性质，都是敌人。<br>
<br><br>
炎症，就是苍蝇。我们要做的不是打苍蝇，而是改变吸引苍蝇的环境。
:::

::: info 思考二
再来说说补肾。
炎症是细菌，细菌是寒凉也喜寒凉。耳朵招细菌了，说明耳朵的温度低了，也就是肾系统温度低了。所以所有的炎症，都是寒症。<br>
炎症大多伴随发烧，所以人们常常认为是热症。发烧是身体为驱赶细菌而强行升温，升温需要消耗资源，升的是表温，消耗的是内部资源，所以发烧使里寒更甚。<br>
治寒的办法，最直接就是：灸。所以在《灸绳》中，治炎症高烧，都是大胆用灸，每获奇效。<br>
治中耳炎灸哪里？《灸绳》中提到，点灸耳尖有奇效。从补肾的角度，也可灸身柱。
:::

::: tip 案例三	急性耳聋
病因分析：聋，是瘀堵；耳，是心兼肾。<br>
治疗方案：放血（中指、小指四缝放血）
:::

::: info 小结	
如果是心火，拍肘；<br>
如果是瘀堵，放血。放血必须要放对地方，否则是无效的。<br>
周老师的方法，一般急性外因引起的瘀堵，用手指四缝放血；如果内因引起的瘀堵，用胆胃放血。
:::


## 心开窍于目，久视伤血劳于心

心开窍于目，主要在眼角，因为小肠经走眼头和眼尾。

::: tip 案例一	飞蚊症
眼角有黑影，其原因是眼底血管有血栓。<br>
治疗方案：揉心经，拨心经，在整条手臂上揉，把心经上的结揉开，15分钟，就好了。<br>
揉脖子，左右两边都要揉。揉脖子好得更快。
:::

::: tip 案例二	结膜炎，眼角红
吃龙胆泻肝丸、黄连羊肝丸，都不好。因为不是肝的问题，是心的问题。<br>
治疗方案：清心明目上清丸<br>
放血，放血的效果来得更快
:::

::: info 思考
笔者因爬山受风加生气，右眼外眼角痛。吃了两天龙胆泻肝丸，不愈反重。买珍珠明目滴眼液，日滴七八次，第二日就好了。
:::


## 心主血脉（附实操）

所有脉络上的问题都是心脏病。但凡脉络上看到有深色瘀堵的血管（在腿上、胳膊上都可以看到），都是心脏的问题，治疗办法就是在瘀堵处放血。
这是以疏为补，帮助心脏恢复正常工作。另一个办法是揉脖子，左心右肝。<br>
揉脖子和放血，都是针对血脉的治疗，是治心的主要方法。

::: tip 案例一	大腿脾经瘀络
大腿脾经上有非常黑的瘀络，黑络处放血，流了很多血。放血之后，他吃了饭就困的毛病没有了。<br>
病因分析：因为血液涌向胃导致脑部缺血犯困，这是由于心脏动力不足了，所以是心的问题。所以一放血马上就好。<br>
另一方案：金匮肾气丸<br>
方案依据：治心不治心，治肾。心有问题是因为肾的原动力不足了。吃一段时间就好了
:::

::: tip 案例二	脉管炎
如果按消炎的方式治，越治越坏，因为消炎药都是寒凉的。脉管炎本就是大寒之症。一老头，70多，脉管炎，大夫让锯腿，他不同意。<br>
周老师的同学给他治，吃龙胆泻肝丸，加按摩（隔两天一次），两个多月，好了。<br>
周老师的治疗方案：<br>
放血——在小指、拇指上放血（小指拇指代表下肢）<br>
放血——在胆经瘀堵处放血（心与胆别通）<br>
龙胆泻肝丸（虚则补其母，肝以疏为补，肝功强了，心就好了）<br>
按摩
:::

:::


## 心主神志

是心在想问题，不是脑。<br>
神经病、精神病，都不是脑的问题，是心的问题。<br>

::: tip 神志不清
病因分析：痰迷心窍
:::

病因分析：是心的问题，心脏弱或有病
:::

::: tip 不爱说话、不想说话
病因分析：是心的问题，心脏弱或有病<br>
相反的，如果说话多，是肝郁气滞的表现
:::

::: tip 精神病
病因分析：是心的问题，心脏弱或有病<br>
治疗方案：放血<br>
蒙石滚痰丸
:::

::: info 小结	
一切心理问题、精神的问题，不需要心理咨询，治心就对了。治心的方法：放血，揉脖子，拨心经、小肠经，揉腹股沟。
:::


## 心主汗，汗血同源

::: tip 汗多
这是指非正常出汗多，比如一吃饭就呼呼出汗，或者稍一行动就呼呼出汗。<br>
原因：心阳弱，固不住汗。<br>
治疗方案：治小肠，补心阳
:::

::: tip 糖尿病采用放血疗法
周老师自己因为糖尿病，放血放了20多次，糖尿病是好了很多，可是出汗多了，这是心阳弱的表现，是因为血放多了导致的心阳虚。养了3个月才养好。<br>
思考：为什么糖尿病可用放血治疗法？<br>
周老师的糖尿病是餐后血糖高，是脾病；虚则补其母，治心；心以疏为补，所以放血。
:::

::: info 关于虚实和补泻
虚和实都只是局部的概念。比如心经某一段堵了，痛，堵的地方为实，堵的下游必然为虚。从实的角度，用泻法治，可以；从虚的角度，用补法治，也可以。
:::

::: tip 不出汗
周老师有病人说“我三年没出过汗了，今天你终于给我揉出汗来了”<br>
病因分析：心阴不足。今天把经络揉通了，所以汗出了<br>
治疗方案：补心阴。把堵的地方揉通了，所以堵点的下游就有了新的气血滋养，功能就恢复正常了。
:::

::: info 关于补泻
补，有几种补法。<br>
吃食药为补。这是普补全身；<br>
按揉为补。把堵的地方揉通了，堵点的下游有了新的气血滋养，这就是补了局部了；<br>
放血为补。这是同样的道理，把堵的地方放通了，堵点的下游有了新的气血滋养，这就是补了局部了。
:::

两颐长疙瘩——肾上有热，两颐赤。所以这是肾上有热的表现。腘窝放血<br>

::: tip 面部美容
治疗方案：治心。揉脖子，把脖子揉开
:::

::: tip 满脸褶子
这是心脏有病<br>

一女，40多岁，中医主任，脸黑5年，到处治，治不好。<br>

:::


## 诸痛痒疮缘于心

### 痛

所有的痛，都是心的问题，不是神经的问题。<br>
痛，最好的治疗方法就是放血，放血的止痛效果是立竿见影的。<br>
放血要放对地方，放血要放够量。<br>
痛的另一个治疗方法是揉脖子（脉疗法）。

::: tip 胳膊痛
周老师的学生收了一病人，胳膊痛，痛得不能入睡，疼了七八天了，到处治，西医、针灸、拔罐、按摩，都治不好。<br>
学生问：怎么治？周答：放血。问：放哪儿？答：等我来看看。<br>
周老师让病人撩开裤腿，看见在小腿胆经上有一条黑黑的瘀络，该处放血。血一出，胳膊一点都不疼了。<br>
方案依据：上病下治（胳膊——腿）<br>
心与胆别通（心的问题，放胆经的血；胆的问题，放心经的血）
:::

::: tip 浑身胀。左侧手臂、腿疼麻了5年
一刺络拔罐同行，因浑身胀得难受，来求治。周老师给他全身揉的时候，看他左侧小腿胆经上有黑黑的瘀络，就在此处放血。一放，血像喷泉一样，先喷后流，后来不流了，但病人一动就又流，这样一直流了40分钟才停止。流血期间，周老师说止止血吧，病人说不要，很舒服，病侧胳膊和腿都很舒服。原来他左胳膊疼了5年，同侧腿也疼麻了5年，像半身不遂一样。放血以后，他说现在病侧比好的一边还轻松。接着他要求周老师在他好的一边放血，这次就只流了一点点，也没什么效果。<br>
方案依据：上病下治（胳膊——腿）<br>
心与胆别通
:::

::: info 小结	
放血要放够量。放血要让血自然流，自然止。40分钟、1小时都可以，以患者感觉舒服为度，人体有自我修复机制，坏血出完了，就会自动止血。
:::

::: tip 脖子痛
快50岁的针灸科大夫，脖子痛了15年<br>
周老师观察其小腿上有很多黑黑的瘀络，胆经、膀胱经都有，哪有瘀络就放哪，一放，脖子就不疼了。<br>
第二次又疼又来放血，结果放完还疼，周老师问徒弟放的哪里？徒弟答：大椎、胳膊肘、手指都放了。<br>
周老师说地方没放对，下午又在小腿上找瘀络放，一放就好了。<br>
方案依据：上病下治（脖子——腿）<br>
心与胆别通
:::

::: info 小结	
放血要放对地方。不同的症状要在相应的地方放血，放对了地方，一针下去血一出来，立刻止痛，没放对地方，则不会起效。
:::

::: tip 三叉神经痛
三叉神经痛，是在两颐的位置，是肾经末梢的地方。<br>
病因分析：在肾经上，是肾的问题。肾水浅了，不足以制火，肾火跑出来了，上行到肝，引动肝火（雷火），肝肾火再上行，直达肾经末梢，引起了三叉神经痛<br>
治疗方案：扎同侧肾关、太溪和对侧太冲。扎了不到5分钟，就不疼了<br>
方案依据：同侧肾关、太溪——补肾水； 补，在同侧补。<br>
对侧太冲——疏肝； 疏，在对侧疏。<br>
肾关，是董穴中补肾第一要穴<br>
太溪、太冲是原穴<br>
太冲对于肾引起的痛有着立竿见影的功效——实则泄其子<br>
周老师大多只用原穴和董穴<br>
上病下治（头——下肢）
:::

::: tip 风湿性心脏病
浑身关节疼<br>
周老师揉脖子，揉了20分钟就好了
:::

::: tip 揉宗筋很疼	
有学员说宗筋不好用，揉着太疼了。越疼，说明心脏越不好，一定要揉到不疼。
:::

::: tip 笔者头痛
头上部里面痛，痛得想吐。<br>
揉脖子，揉了20多分钟，不想吐了，头痛也好了大半。<br>
分析：头上部里面痛，是巅顶痛，肝引起的；痛得想吐，肾为胃之关，是肾引起的；揉脖子，治心，同时泻了肝补了肾。
:::


### 痒

所有的痒，都是心脏的问题。<br>
痒的最佳治疗方案：放血、揉脖子。<br>

::: info 特殊：老年性皮肤痒
特殊：老年性皮肤痒，这是血虚生风的表现，不适合放血和揉脖子，吃柏子养心丸、安神补心丸。
:::

::: tip 胳膊痒	
一老头，胳膊痒。虽然长年在周老师处按摩，但是不相信中医疗法<br>
周老师叫放血，放了一点点，老头说不放了不放了，结果痒只止了一点。然后上医院检查，查血，抽了27管血，抽完胳膊就不痒了。<br>
原因：因为医院抽血是在肘窝，这是心包经的位置，心上有邪沉于两肘。
:::

::: info 放血要点
放血要放对地方；放血要放够量。只要放对了地方，放够了量，效果是立竿见影的。
:::

::: tip 手背痒

::: info 柏子养心丸（补气、补血、安神）

### 疮

所有的疮，都是心里的毒火。长出疮来，是好事，这是毒火发出来了，毒火有了出处，它就不会积在身体里为害。所以，对于疮，千万不能消炎，因为一消炎，就把毒火憋回去了，它在身体里乱走，不知道又走到什么地方为害。中医西医方式的消炎都不行，膏、药都不要用，等它长，长熟了，就出脓了，出脓之后就好了。如果不放心，再刺络拨个罐，把余脓拨出来。成脓之后可用拨毒膏帮助拨毒。<br>
对待疮的最正确方法：<br>
不消炎<br>
不清热<br>
等它长<br>
等它长熟<br>

::: tip 脸上长疮
一男脸上长疮，他妻子是中医中任，给他吃药清火消炎，疮好了，留下个疙瘩又疼又痒。到北京协和医院看，医生说是脸癌，需要手术治疗。改投周老师治下。<br>
周老师给他捏了两次，就出脓了，然后用拨毒膏，拨完就好了。<br>
注意：一定要等它长熟，出脓了就是长熟了。再用拨毒膏或刺络拨罐。
:::

::: tip 肘窝长疮
明朝一名医，去拜师，三年学徒，到两年半的时候，师傅突然要出诊一个月，嘱他守家照顾师妹。原来师妹有病，师傅自知治不好，不忍看女儿不治而死，故避走。名医一看，有办法，告知师妹，你有毒火在心，肘窝将起一疮。果然三天后，肘窝长出一疮，不管它，让它长，等长熟了，用拨毒膏，拨完毒再吃点药，好了。
:::

::: tip 周老师自己连续三年长过三次大疮
一次是胸口长一疮。这是心、任脉之火。不打针不吃药不管它，让它长，长熟了，出脓了，把脓挤掉就好了。不放心的话可以再刺络拨个罐。拨完用创可贴一贴，第二天就好了。<br>
一次是大腿根长一疮。这是心、肝之火。同上法。<br>
一次是肛周长一疮，这次有点要命，走路痛，后来连尿都尿不出了。还是不打针不吃药不擦药不管它，让它长，等它熟了，脓就出了，在脓疮周围刺络拨罐，连脓带血拨干净，拨完贴一创可贴，当天就不痛了，第二天封口，第三天痊愈了。<br>
周老师第一次肛周长疮，因为急痛攻心乱了方寸，上医院，开刀，吃药，弄了一个多月才好，好是好了，落下个肛瘘。又做手术，把肛瘘切除了。折腾了很长时间。所以第二次周老师长经验了，用自己的方法对待疮，结果三天就痊愈了。
:::
